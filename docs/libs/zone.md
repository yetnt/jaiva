### `tz_getAll <- `_**`[]`**_
 
The complete list of IANA format timezone constants.
 
**Example:**
```jaiva
tsea "jaiva/timezone"!

@ Get all timezone constants
maak list <- tz_getAll!
@ Print everything with Etc prefix
colonize item with list ->
    if (item ? "Etc") ->
        khuluma(item)!
    <~
<~
```
